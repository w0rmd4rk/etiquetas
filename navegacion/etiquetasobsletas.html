<center>

Obsoleta

Texto centrado (no usar)

Alternativa moderna: Usar CSS text-align: center

<font>

Obsoleta
<font color="red" size="4" face="Arial">Texto con formato</font>
Texto con formato (no usar)

Alternativa: Usar propiedades CSS color, font-size, font-family
