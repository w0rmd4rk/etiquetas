Guía Completa de Meta-Información y Recursos HTML

Etiquetas Meta Esenciales

Charset Declaration

<meta charset>
<meta charset="UTF-8">

Define la codificación de caracteres del documento.

Viewport

<meta name="viewport">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Controla el viewport para responsive design.

Optimización SEO

Meta Descripción

<meta name="description">
<meta name="description" content="Descripción única de 150-160 caracteres">

Open Graph (Social Media)

<meta property="og:">
<meta property="og:title" content="Título para redes sociales"> <meta property="og:image" content="imagen-social.jpg"> <meta property="og:url" content="https://tusitio.com">

HTML Semántico

<header role="banner"> <nav aria-label="Navegación principal"> <main role="main"> <article> <section aria-labelledby="titulo-seccion"> <footer role="contentinfo">
